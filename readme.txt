FRANCAIS
Bonjour et bienvenue sur Pokemon Titane!
Ce projet a été réalisé en 2 mois par Matthieu Sauer, Yannick Lafosse, Michael Bernard et Alexandre Seignol.
Pour ce projet, notre tâche était de réaliser un RPG de toute pièce grâce à la librairie Graphique CSFML.
Nous devions implémenter un déplacement, des quêtes, des PNJ (personnages non joueurs), des ennemis, un inventaire, etc...
Nous avons décidé de recréer un Pokemon, baptisé Pokemon TITANE.
Le jeu est encore en développement, et nous continuons d'améliorer les fonctionnalités disponibles.
Nous espérons que notre jeu vous plaira,
Cordialement,
Alexandre, Michal, Yannick et Matthieu

ENGLISH
Hello and Welcome on Pokemon Titanium!
This project was created during 2 months by Matthieu Sauer, Yannick Lafosse, Michael Bernard and Alexandre Seignol.
We had to create and RPG thanks to the Graphical library Named CSFML.
The main parts of this project were to create a moving character, some quests, some NPC (Non-Playable character), ennemis, inventory, etc...
We decided to create a Pokemon, called Pokemon Titanium.
The game is already in developpment, and we continue to improve it.
We hope you will enjoy our game,
Regards,
Alexandre, Michael, Yannick and Matthieu
