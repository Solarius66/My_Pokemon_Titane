Bienvenue sur Pokémon Titane,

Votre quête est d'amener le Pokémon principal que vous aurez choisi devant le prof Jérém au niveau 20.
Vous rencontrerez pour cela des Pokémons dans les hautes herbes que vous devrez combattre.
Vous pourrez régénérer la vie de vos Pokémons dans les centres Pokémons de chaque ville, ce sont des bâtiments avec un toit orange.
Plus vous avancerez dans le monde, et plus le niveau des Pokémons que vous affronterez sera élevé.
Vous rencontrerez également des Pokémons différents en fonction de la zone dans laquelle vous vous trouverez.
Bonne chance.


Ce jeu est en version bêta, la capture des Pokémons, et les personnages du jeu ne sont pas encore intégrés.

Si vous souhaitez tester ce jeu, des parties sauvegardées ainsi que leurs descriptions sont présentes dans le dossier save.
Pour les utiliser il suffit de placer le fichier .save du dossier dans le dossier où est exécuté le binaire.
